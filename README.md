# Rodinné financie 💰

PWA aplikácia na sledovanie rodinných financií.

## Tech stack
- React 18 + Vite 5
- TypeScript
- Dexie.js (IndexedDB — lokálne ukladanie dát)
- Tailwind CSS
- Recharts

## Spustenie lokálne
npm install
npm run dev

## Build
npm run build
npm run preview

## Deploy
./deploy.sh

## Funkcie
- 📊 Prehľad príjmov a výdavkov
- 💳 Variabilné výdavky s kategóriami
- 🔒 Fixné výdavky
- 🎯 Rozpočtové limity per kategória
- 📈 Grafy a štatistiky
- 💾 Export / Import dát (JSON)
- 📱 PWA — inštalovateľné na mobil
- 🔒 Privacy-first — všetky dáta zostávajú lokálne v prehliadači
